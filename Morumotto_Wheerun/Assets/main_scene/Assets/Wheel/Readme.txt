『回し車』

VRchat向けですが特に用途の制限はないので
下記を参考にご自由にお使いください。

有料の方は回転させるシステムが付いてます。
降りた後は横のスイッチで停止させてください。
またFANBOX500円プラン「400円以下のもの使い放題」プランの対象商品となります。

	AlbedoTransparency.png :アルベドマップです。standardシェーダーとか使う時に使います。
	Normal.png : ノーマルマップです。ノーマルマップに対応したシェーダーに入れると質感が出ます。
	Metallic.png : メタリックラフネスマップです。メタリックwマップに対応したシェーダーに入れると質感が出ます。
	Roughness.png : ラフネスマップです。ラフネスマップに対応したシェーダーに入れると質感が出ます。
